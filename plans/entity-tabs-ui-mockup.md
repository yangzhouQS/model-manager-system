## 二、SINGLE（单表）— 无标签页

实体数量 = 1，不显示标签页区域，直接展示字段配置表格。

```
┌─ 字段配置 ─────────────────────────────────────────────────────────────┐
│                                                                        │
│  ┌───┬────────┬────────┬────────┬────────┬────────┬──────────────────┐ │
│  │ # │字段名  │别名    │SQL类型 │TS类型  │来源    │ ...              │ │
│  ├───┼────────┼────────┼────────┼────────┼────────┼──────────────────┤ │
│  │ 1 │id      │主键    │BIGINT  │string  │数据库  │                  │ │
│  │ 2 │name    │名称    │VARCHAR │string  │手动    │                  │ │
│  └───┴────────┴────────┴────────┴────────┴────────┴──────────────────┘ │
└──────────────────────────────────────────────────────────────────────────┘
```

**说明：** 和当前行为一致，无变化。

## 四、MASTER_CHILD（一主多子）— 分组标签页

实体数量 = 1（主表）+ N（子表），标签页分为两组，用分隔线和组标题区分。

│  │ │ 主表 ⑤  │  ← 蓝色激活态，带下划线                               │  │
│  │ └──────────┘                                                      │  │
│  │ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─   │  │
│  │ 子表                                                               │  │
│  │ ┌──────────┐  ┌──────────┐  ┌──────────┐                         │  │
│  │ │ 子表1 ③ │  │ 子表2 ④ │  │ 子表3 ② │                         │  │
│  │ └──────────┘  └──────────┘  └──────────┘                         │  │
│  └──────────────────────────────────────────────────────────────────┘  │
│                                                                        │
│  ┌───┬────────┬────────┬────────┬────────┬────────┬──────────────────┐ │
│  │ # │字段名  │别名    │SQL类型 │TS类型  │来源    │ ...              │ │

**视觉细节：**
- 两组之间用虚线分隔
- 每组左侧有灰色小标题「主表」/「子表」

- 子表标签左侧带一个小圆点（🟢 绿色）作为层级标识
- 选中态：蓝色文字 + 蓝色底部边框
- 数字角标：灰色圆角背景，显示字段数量

## 五、MASTER_CHILD_GRANDCHILD（主子孙）— 三级分组标签页

## 六、样式规范

### 颜色方案

| 元素 | 颜色 | 说明 |
|------|------|------|
| 组标题文字 | `#909399`（灰色） | 12px，小标题 |
| 分隔虚线 | `#e4e7ed`（浅灰） | 1px dashed |
| 主表标签 | 默认样式 | 无特殊标记 |
| 子表圆点 | `#67c23a`（绿色） | 6px 圆形 |
| 孙表圆点 | `#409eff`（蓝色） | 6px 圆形 |
| 激活态 | `#409eff`（蓝色） | 文字 + 底部边框 |
| 字段数角标 | 激活时蓝底蓝字，非激活时灰底灰字 | |

### 布局参数

| 属性 | 值 |
|------|------|
| 标签页区域背景 | `#fafafa` |
| 标签页内边距 | `0 16px` |
| 组标题字号 | `12px` |
| 组标题上边距 | `8px`（第一组无） |
| 标签项间距 | `0`（平铺） |
| 标签项内边距 | `8px 16px` |
| 圆点大小 | `6px` |
| 圆点与文字间距 | `6px` |

## 七、关联关系配置区域（多实体通用）

**字段下拉选项增强：**

```
源字段下拉（选择源实体为"子表1"后）：
┌───────────────────────┐
│ org_id      (string)  │
│ dept_id     (number)  │
│ name        (string)  │
│ code        (string)  │
└───────────────────────┘

目标字段下拉（源字段选了 dept_id (number) 后，自动过滤兼容字段）：
┌───────────────────────┐
│ id          (string)  │  ← number→string 兼容 ✅
│ dept_no     (string)  │  ← number→string 兼容 ✅
│ dept_code   (string)  │  ← number→string 兼容 ✅
│ count       (number)  │  ← 同类型兼容 ✅
│ is_active   (boolean) │  ← 不兼容，已过滤 ❌
└───────────────────────┘
```

**类型不兼容警告样式：**

```
┌─────────────────────┐     ┌─────────────────────┐
│ flag (boolean)    ▼ │ ──→ │ status (number)   ▼ │  ⚠️ 类型不兼容: boolean ≠ number
└─────────────────────┘     └─────────────────────┘
                              ↑ 红色文字提示
```

## 八、实体结构 → 标签页展示对照表

| 实体结构 | 标签页展示 | 分组 | 圆点颜色 |
|---------|-----------|------|---------|
| SINGLE | 无标签页 | — | — |
| EXTENDED_SINGLE | 平铺标签页 | 不分组 | — |
| MASTER_CHILD | 分组标签页 | 主表 + 子表 | 子表：绿色 |
| MASTER_CHILD_GRANDCHILD | 分组标签页 | 主表 + 子表 + 孙表 | 子表：绿色，孙表：蓝色 |
